체크리스트 사용 안내

📋 체크리스트 작성 전 알아두세요

  • 강아지 입양은 10-15년 이상의 장기적인 책임이 필요해요.
  • 모든 가족 구성원의 동의가 필요한 중요한 결정이에요.
  • 정직한 자기 평가가 성공적인 입양의 첫걸음이에요.

✅ 체크리스트 사용 방법

  1. 각 항목을 꼼꼼히 읽고 현재 상황에 맞게 체크해주세요.
  2. 궁금한 점이 있다면 물음표(?) 버튼을 클릭해서 자세한 설명을 확인하세요.
  3. '결과 확인하기' 버튼을 눌러 준비 정도를 확인하세요.
  4. 결과를 PDF로 저장할 수 있어요.
