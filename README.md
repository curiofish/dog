# 강아지 입양 준비도 체크리스트

강아지 입양을 고려하는 분들을 위한 온라인 체크리스트입니다.

## 주요 기능

- 시간 관리, 책임감, 재정적 책임, 생활 환경, 장기적 계획 등 5가지 영역 평가
- 체크리스트 작성 후 준비도 점수 확인
- 맞춤형 조언 제공
- PDF 저장 기능

## 사용 방법

1. 각 항목을 꼼꼼히 읽고 현재 상황에 맞게 체크
2. 궁금한 점이 있다면 물음표(?) 버튼을 클릭하여 자세한 설명 확인
3. '결과 확인하기' 버튼을 눌러 준비도 확인
4. 필요한 경우 PDF로 저장
