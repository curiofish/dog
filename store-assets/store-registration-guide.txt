구글 플레이 스토어 등록 가이드

1. 필요한 계정
- Expo 계정 (fishcurio@gmail.com)
- 구글 플레이 스토어 개발자 계정 ($25 등록비)

2. 앱 정보
- 앱 이름: 강아지 입양 체크리스트
- 패키지 이름: com.curiofish.dogchecklist
- 버전: 1.0.0
- 최소 안드로이드 버전: 6.0 (API 23)

3. 필요한 이미지
- 앱 아이콘 (512x512 PNG)
- 스크린샷 (최소 2장)
- 피처드 그래픽 (1024x500 PNG)

4. 등록 절차
1) Expo 계정으로 로그인
2) eas build:configure 실행
3) eas build --platform android 실행
4) 구글 플레이 스토어 개발자 콘솔에서 새 앱 생성
5) 앱 번들 업로드
6) 스토어 등록 정보 입력
7) 개인정보 처리방침 URL 등록
8) 출시 검토 요청

5. 주의사항
- 앱 아이콘과 스크린샷은 최신 디자인으로 준비
- 개인정보 처리방침은 웹 호스팅 필요
- 앱 설명과 스크린샷은 한글/영문 모두 준비
