강아지 입양 준비 체크리스트

강아지 입양을 고려하는 분들을 위한 체크리스트 앱입니다.

주요 기능:
• 시간 관리, 책임감, 환경, 준비사항 등 4가지 영역 평가
• 각 항목별 상세 설명 제공
• 체크리스트 작성 후 준비도 점수 확인
• 맞춤형 조언 제공

사용 방법:
1. 각 항목을 꼼꼼히 읽고 현재 상황에 맞게 체크
2. 궁금한 점이 있다면 물음표(?) 버튼을 클릭하여 자세한 설명 확인
3. '결과 확인하기' 버튼을 눌러 준비도 확인
4. 점수에 따른 맞춤형 조언 확인

이 앱을 통해 강아지 입양 전 필요한 준비사항들을 꼼꼼히 점검해보세요.
